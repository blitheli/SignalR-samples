
创建一个Asp.net Core Web程序，演示SignalR的通信

	本项目创建了一个Hub服务器，同时创建了一个简单的网络页面，在页面中输入字符串
可通过Hub服务器转发到所有的网页中。


发布本程序
-----------------------------------------------------------
右键工程"ChatSample"，选择"发布"

命令行运行
-----------------------------------------------------------
将发布后的文件夹拷贝到网络服务器上
在程序文件夹地址栏输入cmd
运行: dotnet ChatSample.dll --urls http://*:8003
端口号8003可以修改

查看
-----------------------------------------------------------
打开网络内任意终端浏览器
输入: http://Ip:8003


项目结构
-----------------------------------------------------------
服务端:	
	hub连接地址:		/receivemessagehub
	函数(客户端调用):	Send(message)
						向所有客户端发送此字符串

客户端:
	网页打开后，输入字符串，点击发送后
	调用客户端Send函数，发送字符串
